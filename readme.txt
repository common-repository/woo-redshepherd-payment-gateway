== Description ==
This is the Red Shepherd Payment Plugin for WooCommerce. This will allow you to securely collect Credit Card / Debit Card and ACH (Pay by Check) payments on your WooCommerce Check-out page.

Visit [Plugin Documentation](https://www.redshepherd.com/woocommerce.html) for more details and documentation.

== Installation ==
1. Upload the entire 'woo-redshepherd-payment-gateway' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
